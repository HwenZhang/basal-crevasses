Code description

This folder contains the code to predict the quasi-static path of a basal crevasse using boundary element method. For details about the BEM code please refer to https://github.com/Timmmdavis/CutAndDisplace.

How to run it?

* Run the script "AddFilePaths.m" in the folder "StickyPatchFracture".

* Load the script "MainFrame2DFractureGrowingFracNVD.m" in matlab and set up the values of geometric parameters and material properties between line 26 and 57. The run the script.
